Execution

Servlets exercise -

    The purpose of the exercise is to build a part of a site in stackoverflow style.
    The site displays a list of questions that will be stored in the file.
    Users can read questions and answers, and answer questions.
    The site displays a list of questions and next to each question, the number of answers to it.
    You can click on the (button) question and the site shows all the answers to the question.
    A user can add an answer to a question.

Assumptions

Jsoup library, org.glassfish:javax.json:1.1.14
